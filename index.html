Scryfall to Shopify Exporter

How to use:

  1. Export your collection from Manabox as a CSV file
  2. Upload the Manabox CSV file using the button below
  3. The tool will automatically fetch Scryfall data and export to Shopify format
  4. Two CSV files will be downloaded: shopify_products.csv and shopify_inventory.csv

Note: This tool expects the standard Manabox CSV format with columns: Name, Set code, Set name, Collector number, Foil, Rarity, Quantity, etc.
